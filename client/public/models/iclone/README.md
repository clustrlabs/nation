# iClone Models Directory

This directory contains high-quality iClone human models for the loading screen animation.

Supported formats:
- .fbx
- .obj
- .gltf / .glb

Place your iClone models in this directory using the following structure:
```
iclone/
  ├── humans/
  │   ├── male/
  │   │   └── model.fbx
  │   └── female/
  │       └── model.fbx
  └── animations/
      └── idle/
          └── animation.fbx
```
